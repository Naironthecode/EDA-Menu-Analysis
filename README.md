## 📌 Project Description

This project focuses on performing Exploratory Data Analysis (EDA) on a menu dataset to understand nutritional patterns and calorie distribution across different food categories. 

The analysis includes data cleaning, statistical exploration, visualization, and correlation analysis to extract meaningful insights. Various graphs such as bar plots and heatmaps are used to identify trends and relationships between features like calories, fat, sugar, and protein.

The goal of this project is to gain insights into food composition, identify high and low calorie items, and understand how different nutritional factors are related.
